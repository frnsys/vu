# `vu`

A super simple image viewer. It just shows images and plays gifs.

Usage:

```bash
vu image-01.jpg image-02.jpg
```

Use `vu --help` for more info.

Shortcuts:

- `f`: Toggle Fullscreen
- `e`: Zoom in
- `h`: Zoom out
- `Up/Right/Down/Left`: Pan
- `'`: Next image
- `,`: Prev image
- `i`: Toggle info
- `q`/`Esc`: Quit
```
